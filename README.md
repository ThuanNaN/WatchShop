# React E-commerce App - Watch shopping

We use ReactJS to build Fronend and ExpressJS for Backend/Api

Requirements:

- NodeJS: 16.16.0 LST
- ReactJS
- Redux toolkit

## Usage

1.Install backend package and start service:

```bash
npm i
npm run dev
```

2.Install frontend package and start:

```bash
cd frontend
npm i
npm start
```
